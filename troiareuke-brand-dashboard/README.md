# 트로이아르케 브랜드 대시보드

## 현재 상태

이 대시보드는 이제 단순 샘플 화면이 아니라,
공개 웹페이지에서 확인한 실제 출처 기반 스냅샷 데이터를 포함합니다.

기준일:

- 2026-04-14

## 현재 들어간 실제 데이터 종류

- 트로이아르케 공식몰 리뷰 노출 수
- 트로이아르케 관련 기사 모멘텀
- 달바 화해 리뷰 데이터
- 닥터멜락신 기사 기반 매출 데이터
- 톰프로그램 기사 기반 커머스 모멘텀
- 메디테라피 기사 기반 누적 판매량
- 파메스테틱 공개 판매 페이지 정보

## 파일 설명

- `index.html`: GitHub Pages 배포용 화면 구조
- `styles.css`: 반응형 UI 스타일
- `app.js`: 실제 스냅샷 데이터, 필터, 렌더링, 실시간 API 연결 지점
- `GITHUB_PAGES_DEPLOY.md`: 배포 가이드

## 중요한 구분

현재 모드 2개:

1. 실제 공개 스냅샷
- 웹에서 확인한 실제 공개 데이터를 화면에 반영
- 기간 변경을 해도 자동 재집계되지는 않음

2. 실시간 연동
- API 연결 시 기간별 실제 재집계 가능

## 왜 아직 완전 실시간이 아닌가

GitHub Pages는 정적 호스팅이라서
네이버 검색량, 구글 검색량, 멘션 수집을 직접 실시간 처리하기 어렵습니다.

그래서 실시간 운영 구조는 아래가 필요합니다.

- GitHub Pages: 화면
- 별도 API: 데이터 수집과 집계

## 실시간 연결 포인트

`app.js`

```js
const LIVE_API_CONFIG = {
  enabled: false,
  endpoint: "/api/dashboard"
};
```

이 부분을 실제 API 주소로 바꾸면 됩니다.
