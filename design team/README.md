# Design Team Workspace

This folder is the working home for the beauty brand design team.

## Folder Map

- `references/troiareuke/images`
  - Brand references, detail page captures, mood boards, product images, packaging images
- `references/troiareuke/texts`
  - Product info, copy references, brand rules, prohibited claims, launch notes
- `outputs/detail_pages`
  - Final detail page drafts, slide-by-slide plans, visual direction docs
- `outputs/copy`
  - Copywriting drafts, headline options, CTA options, script drafts
- `templates`
  - Reusable working templates for TROIAREUKE detail page production

## TROIAREUKE Working Rules

- Canvas unit: `1080 x 1350px`
- Actual detail-page safe width: `860px`
- Font: `Pretendard`
- Weight pair: `Light + Semibold` or `Regular + Bold`
- Tracking: `-20 / -40 / -60`
- Width scale: `98%`
- Alignment: left aligned by default
- Layout: maintain generous whitespace
- Style: minimal, premium
- Principle: one image equals one message
- Avoid information overload

## Recommended File Names

- `product_name_detail_plan.md`
- `product_name_copy_v1.md`
- `product_name_reference_notes.md`
- `product_name_slide_01.png`
